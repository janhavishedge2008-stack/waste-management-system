                    WHERE IS THE ADMIN PAGE?
================================================================================

The admin page is now VERY VISIBLE in the navigation bar!

================================================================================
STEP-BY-STEP GUIDE:
================================================================================

STEP 1: Start the Server
-------------------------
Run: START_SERVER.bat

Or manually:
    cd C:\Users\LENOVO\Desktop\new
    venv\Scripts\activate
    python manage.py runserver

STEP 2: Open Website
---------------------
Go to: http://localhost:8000

STEP 3: Login as Admin
-----------------------
1. Click "Login" button (top right corner)
2. Enter:
   Username: admin
   Password: admin123
3. Click "Login"

STEP 4: Find Admin Button
--------------------------
After logging in, look at the TOP NAVIGATION BAR.

You will see a YELLOW/ORANGE button that says:
    🛡️ Admin

It's located between "Company Tie-Up" and your username dropdown.

STEP 5: Click Admin Button
---------------------------
Click the yellow "Admin" button.

You'll be taken to: http://localhost:8000/admin/

ALTERNATIVE WAYS TO ACCESS ADMIN:
================================================================================

Method 1: Direct URL
--------------------
http://localhost:8000/admin/

Method 2: From Username Dropdown
---------------------------------

Method 3: From Blog Page
-------------------------
1. Go to Blog page
2. Click "Add New Blog Post" button
3. Takes you to admin

================================================================================
WHAT IF I DON'T SEE THE ADMIN BUTTON?
================================================================================

Reason 1: Not Logged In
------------------------
Solution: Login first with admin/admin123

Reason 2: Not Logged In as Admin
---------------------------------
Solution: Make sure you're using the admin account
         Username: admin
         Password: admin123

Reason 3: Server Not Running
-----------------------------
Solution: Run START_SERVER.bat

Reason 4: Browser Cache
------------------------
Solution: Press Ctrl+F5 to hard refresh
